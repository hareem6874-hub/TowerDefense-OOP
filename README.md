# TowerDefense-OOP-Tower Defense Game (SFML)

This is a simple Tower Defense game made in C++ using SFML library.  
It was developed as a semester project.

---

## Features

- Different types of enemies (Basic, Fast, Tank, Flying, Split)
- Different towers (Cannon, Machine Gun, Sniper, Slow, Fire)
- Tower placement system
- Tower upgrade system (Right click upgrade)
- Enemy wave system
- Score system with high score saving
- Background music support
- Simple UI and health system

---

## How to Play

- Click on tower buttons to select a tower
- Left click on map to place tower
- Right click on a tower to upgrade it
- Enemies come in waves
- Stop enemies before they reach the end
- Each kill gives gold and score

---

## Controls

- Left Mouse Click → Select / Place Tower
- Right Mouse Click → Upgrade Tower
- 1-5 Keys → Select Towers
- ESC / Close Button → Exit Game

---

## Scoring System

- Killing enemies gives score
- Remaining gold adds to score
- Surviving waves increases score
- Highest score is saved in `highscore.txt`

---

## Requirements

- C++ Compiler
- SFML Library (Graphics, Window, System, Audio)

---

## How to Run

1. Install SFML
2. Compile all `.cpp` files with SFML linked
3. Make sure all `.h` files are in same folder
4. Run the executable
